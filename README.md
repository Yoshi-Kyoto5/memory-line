# Database creation 
## users-table  
|Column|Type|Options|
|------|----|-------|
|nickname|string|null: false, unique: true|
|email|string|null: false, unique: true|
|password|string|null: false|
|encrypted_password|string|null: false|
### Association 
- has_many  :messages
- has_many  :comments

## messages-table 
|Column|Type|Options|
|------|----|-------|
|text|text|null: false|
|image|string|
### Association 
- belongs_to :user
- has_many   :comments

## comments-table 
|Column|Type|Options|
|------|----|-------|
|text|text|
|image|string|
|user_id|integer|
|message_id|integer|
### Association 
- belongs_to :user
- belongs_to :message
